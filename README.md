# Goldn MAX AI
Your personal hybrid Jarvis-style assistant.

## Features
- Offline reasoning mode
- Online GPT-4o-mini mode
- Long-term memory
- Voice input + voice output
- PWA installable app on iPhone

## How to deploy (GitHub Pages)
1. Upload all files to the repo root.
2. Go to Settings → Pages.
3. Enable Pages:  
   - Source: Deploy from a branch  
   - Branch: main  
   - Folder: / (root)
4. Save and wait for GitHub pages link.
5. Open the link on Safari → Add to Home Screen.
